# xiuJiang-downloads（模板）

将本目录内容复制到公开 GitHub 仓库根目录（建议名 `xiuJiang-downloads`），开启 GitHub Pages（Source: GitHub Actions）。

App 默认读取：

```text
https://<github-user>.github.io/xiuJiang-downloads/api/
```

发版步骤见主仓 `docs/自动更新与发版.md`。

**注意：** 不要把 APK 长期 commit 进 git；用 GitHub Releases 托管 APK，JSON 里只写 `apkUrl`。
